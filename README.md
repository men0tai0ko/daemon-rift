> ブラウザ向けタップ系RPG。悪魔を仲魔にし、合体で強化しながら廃都の深層へ潜る放置＋育成ゲーム。

**バージョン:** v0.4.3  
**ステータス:** Phase 5 進行中（最終調整・リリース）  
**ゲームタイトル:** 未確定（仮: DAEMON RIFT / 廃都の悪魔）

## ■ プロジェクト概要

| 項目 | 内容 |
|------|------|
| ジャンル | タップ系RPG（放置＋育成） |

## ■ コアコンセプト

> 「仲魔を集め、合体させ、果てなき廃都の深層へ潜れ」

## ■ 開発の最重要制約

- **1ファイル完結** — 外部ファイル・CDN依存禁止
- **バージョン番号は3箇所同期** — コメントヘッダー / `<meta name="version">` / `APP_VERSION` 定数
- **SAVE_KEYS への追記必須** — 新規フィールドをセーブ対象にする場合は忘れずに追加
