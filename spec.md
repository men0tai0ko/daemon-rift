## ② ゲームコンセプト

### 一言
> 「仲魔を集め、合体させ、果てなき廃都の深層へ潜れ」

### プレイヤーの目的
- より深い階層に到達する（深度記録の更新）
- より強力な上位悪魔を合体で生み出す

### コア体験
1. **放置進行** → 仲魔が自動戦闘で探索を続ける
2. **要所タップ** → 交渉・合体・ボス戦で手動介入
3. **合体ループ** → 弱い悪魔を素材に強い悪魔を生み出す達成感

### 差別化ポイント
- 合体による「捨てる楽しさ」（育てた悪魔を素材にする決断）
- 交渉システムによる会話演出でメガテン世界観を演出
- 放置系でありながら合体戦略に思考要素あり

## ③ ゲームシステム

### 進行フロー
```
タイトル画面
  ↓
ダンジョン探索（放置自動進行）
  ↓ 一定時間ごとに敵エンカウント
戦闘 or 交渉を選択（タップ介入）
  ├─ 戦闘 → 経験値・マッカ獲得
  └─ 交渉 → 仲魔加入 or アイテム獲得
  ↓
5フロアごとにボス出現（手動タップ戦闘）
  ↓
合体ラボで悪魔合体（任意タイミング）
  ↓
仲魔編成を更新 → より深層へ
```

### 操作
- **基本操作:** タップのみ（片手プレイ対応）
- **放置進行:** 探索は自動、プレイヤーは要所で介入
- **介入ポイント:** 交渉選択 / 合体実行 / ボス戦スキル発動

### 勝敗条件
- **継続条件:** 仲魔が全滅しなければ探索継続
- **失敗条件:** 仲魔全滅 → 一定階層前からリスタート
- **目標:** 深度記録（localStorageに保存）の更新

## ④ 悪魔・仲魔システム

### 悪魔データ構造
| フィールド | 内容 |
|-----------|------|
| 名前 | 悪魔名（メガテン風架空名） |
| 種族 | 妖精 / 鬼神 / 堕天使 / 幻魔 など |
| 属性 | 炎 / 氷 / 電 / 物理 / 呪 |
| ステータス | HP / ATK / DEF / SPD |
| スキル | 最大2個（合体時継承可・SKILL.inheritSkill()経由） |
| レベル | 1〜99 |
| レアリティ | C / B / A / S |

### 合体システム
- **2体合体** → 上位悪魔を生成
- 素材の種族組み合わせで生成物が変わる（合体表）
- スキルは素材2体から1個ランダム継承
- 合体後は素材の悪魔が消滅（決断コスト）
- スキル石使用時は確定継承（STATE._skillStoneActive フラグ）

### 交渉システム
- 戦闘中に「話す」を選択
- 選択肢3択（おだてる / 脅す / 金で解決）
- 成功率計算: `base[tactic] + (playerLv - enemyLv) * 0.05`（clamp 0.05〜0.9）
- 成功 → 仲魔加入、失敗 → 逃げられる or 激怒

### 仲魔編成
- 最大3体パーティ
- 自動戦闘時に属性相性が計算される
- 交代・入れ替えは自由（マッカ不要）

## ⑤ 成長・報酬

| 要素 | 内容 |
|------|------|
| 経験値 | 戦闘で獲得 → 悪魔個別レベルアップ |
| マッカ | 通貨。交渉・アイテム購入に使用 |
| マガタマ | 特殊強化素材。ボス撃破で獲得（※実装状況確認要） |
| スキル石 | スキルを合体時に確定継承させるアイテム |

### レベルアップ効果（applyLevelUp）
- maxHp + rand(5,10)、hp = maxHp（全回復）
- atk + rand(1,2)、def + rand(1,2)
- expNext = lv * 20

## ⑥ アイテムシステム

### アイテム一覧（6種）

| アイテム名 | アイコン | 効果 | 最大所持 | 使用タイミング |
|-----------|--------|------|---------|--------------|
| 回復薬 | 🧪 | 仲魔1体のHPを30%回復 | 5 | 探索中・戦闘中（未実装） |
| 万能薬 | 💊 | 仲魔1体のHPを全回復 | 3 | 探索中・戦闘中（未実装） |
| スキル石 | 💎 | 合体時スキルを確定継承させる | 3 | 合体時（フラグ管理） |
| 覚醒の書 | 📖 | 仲魔を即座にLvUP | 2 | 探索中 |
| マッカ袋 | 💰 | マッカを50〜150₪獲得 | 9 | 探索中 |
| 守りの御札 | 🛡 | 次の被ダメージを1回無効化 | 3 | 探索中・戦闘中（未実装） |

### ドロップ率
- 自動討伐時のドロップ率: `0.08 + min(floor * 0.002, 0.12)` = 8%〜20%
- F1〜10: 回復薬・マッカ袋のみ
- F11〜30: 上記 + 万能薬・守りの御札
- F31〜: 全種類

## ⑦ 敵・難易度設計

### 敵の種類
- **雑魚敵:** 各フロアに対応した悪魔
- **エリートモブ:** 5フロアに1体。強化版、交渉不可
- **ボス:** 10フロアに1体。専用スキル・高耐久

### 難易度スケール
- 敵ステータス係数: `1 + Math.log1p(floor-1) * 0.45`
- ボス倍率: 1.9×
- ダメージ計算: `max(1, atk - def*0.5) * 属性倍率 * rand(0.85〜1.15)`

### 属性相性テーブル
| 攻撃＼防御 | 炎 | 氷 | 電 | 物 | 呪 |
|-----------|-----|-----|-----|-----|-----|
| 炎 | 0.5 | 2.0 | 1.0 | 1.0 | 1.0 |
| 氷 | 0.5 | 0.5 | 2.0 | 1.0 | 1.0 |
| 電 | 1.0 | 1.0 | 0.5 | 1.5 | 1.0 |
| 物 | 1.0 | 1.0 | 1.0 | 1.0 | 1.5 |
| 呪 | 1.0 | 1.0 | 1.0 | 0.5 | 0.5 |

## ⑪ ステージ設計

- **構造:** 無限深層（エンドレス）
- **フロア単位:** 1フロア = 3〜5回エンカウント
- **ボス周期:** 10フロアごとに専用ボス
- **記録:** 最大到達深度をlocalStorageに保存

### 深度帯区分
| 深度 | エリア名 | 雰囲気 |
|------|---------|--------|
| 1〜10 | 廃都表層 | チュートリアル相当 |
| 11〜30 | 地下街区 | 中難易度 |
| 31〜60 | 冥界回廊 | 高難易度 |
| 61〜 | 虚無の底 | 超高難易度・無限 |
