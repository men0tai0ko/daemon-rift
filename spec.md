## ② ゲームコンセプト

### 一言
> 「仲魔を集め、合体させ、果てなき廃都の深層へ潜れ」

### プレイヤーの目的
- より深い階層に到達する（深度記録の更新）
- より強力な上位悪魔を合体で生み出す

### コア体験
1. **放置進行** → 仲魔が自動戦闘で探索を続ける
2. **要所タップ** → 交渉・合体・ボス戦で手動介入
3. **合体ループ** → 弱い悪魔を素材に強い悪魔を生み出す達成感

### 差別化ポイント
- 合体による「捨てる楽しさ」（育てた悪魔を素材にする決断）
- 交渉システムによる会話演出でメガテン世界観を演出
- 放置系でありながら合体戦略に思考要素あり

## ③ ゲームシステム

### 進行フロー
```
タイトル画面
  ↓
ダンジョン探索（放置自動進行）
  ↓ 一定時間ごとに敵エンカウント
戦闘 or 交渉を選択（タップ介入）
  ├─ 戦闘 → 経験値・マッカ獲得
  └─ 交渉 → 仲魔加入 or アイテム獲得
  ↓
5フロアごとにボス出現（手動タップ戦闘）
  ↓
合体ラボで悪魔合体（任意タイミング）
  ↓
仲魔編成を更新 → より深層へ
```

### 操作
- **基本操作:** タップのみ（片手プレイ対応）
- **放置進行:** 探索は自動、プレイヤーは要所で介入
- **介入ポイント:** 交渉選択 / 合体実行 / ボス戦スキル発動

### 勝敗条件
- **継続条件:** 仲魔が全滅しなければ探索継続
- **失敗条件:** 仲魔全滅 → 一定階層前からリスタート
- **目標:** 深度記録（localStorageに保存）の更新

### 時間制限
- なし（放置型のため）

## ④ 悪魔・仲魔システム

### 悪魔データ構造
| フィールド | 内容 |
|-----------|------|
| 名前 | 悪魔名（メガテン風架空名） |
| 種族 | 妖精 / 鬼神 / 堕天使 / 幻魔 など |
| 属性 | 炎 / 氷 / 電 / 物理 / 呪 |
| ステータス | HP / ATK / DEF / SPD |
| スキル | 最大2個（合体時継承可） |
| レベル | 1〜99 |
| レアリティ | C / B / A / S |

### 合体システム
- **2体合体** → 上位悪魔を生成
- 素材の種族組み合わせで生成物が変わる（合体表）
- スキルは素材2体から1個ランダム継承
- 合体後は素材の悪魔が消滅（決断コスト）

### 交渉システム
- 戦闘中に「話す」を選択
- 選択肢3択（おだてる / 脅す / 金で解決）
- 敵の種族・HPに応じて成功率変動
- 成功 → 仲魔加入、失敗 → 逃げられる or 激怒

### 仲魔編成
- 最大3体パーティ
- 自動戦闘時に属性相性が計算される
- 交代・入れ替えは自由（マッカ不要）

## ⑤ 成長・報酬

| 要素 | 内容 |
|------|------|
| 経験値 | 戦闘で獲得 → 悪魔個別レベルアップ |
| マッカ | 通貨。交渉・アイテム購入に使用 |
| マガタマ | 特殊強化素材。ボス撃破で獲得 |
| スキル石 | スキルを合体時に確定継承させるアイテム |

### レベルアップ効果
- ステータス上昇（HP/ATK/DEFをランダム振り分け）
- 一定レベルで新スキル習得（最大2スロット）

## ⑥ 敵・難易度設計

### 敵の種類
- **雑魚敵:** 各フロアに対応した悪魔（プレイヤー仲魔にもなれる）
- **エリートモブ:** 5フロアに1体。強化版、交渉不可
- **ボス:** 10フロアに1体。専用スキル・高耐久

### 難易度スケール
- 階層×1.1の倍率でステータス上昇
- 深層ほど属性耐性持ちが増加
- 100階以降は属性無効・吸収持ちが登場

## ⑪ ステージ設計

- **構造:** 無限深層（エンドレス）
- **フロア単位:** 1フロア = 3〜5回エンカウント
- **ボス周期:** 10フロアごとに専用ボス
- **記録:** 最大到達深度をlocalStorageに保存
- **変化要素:** 深度帯ごとに出現悪魔・BGMが変わる

### 深度帯区分（案）
| 深度 | エリア名 | 雰囲気 |
|------|---------|--------|
| 1〜10 | 廃都表層 | チュートリアル相当 |
| 11〜30 | 地下街区 | 中難易度 |
| 31〜60 | 冥界回廊 | 高難易度 |
| 61〜 | 虚無の底 | 超高難易度・無限 |

### STATE 分離ルール
- `[PERSIST]`：`SAVE.save()` で保存される対象。`floor` / `macca` / `kills` / `fusions` / `bestFloor` / `party` / `storage` / `floorProgress`
- `[RUNTIME]`：セッション中のみ保持。`exploring` / `exploreTimer` / `inBattle` / `currentEnemy` / `fusionSlot` / `fusionA` / `fusionB`

### バージョン管理ルール
- `CONFIG.VERSION`・`<meta name="version">`・コメントヘッダー・タイトル画面表示は**すべて CONFIG から自動参照**
- セーブスキーマ変更時は `CONFIG.SAVE_VERSION` をインクリメントし、旧データは自動破棄してユーザーに通知する
